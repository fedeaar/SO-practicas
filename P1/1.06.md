### El sistema operativo del punto anterior es extendido con la llamada al sistema void exec(const char *arg). Esta llamada al sistema reemplaza el programa actual por el c√≥digo localizado en el string (char *arg). Implementar una llamada al sistema que tenga el mismo comportamiento que la llamada void system(const char *arg), usando las llamadas al sistema ofrecidas por el sistema operativo. Nota: Revisar man system, como ayuda.

\
Podemos considerar

```C
void system(const char *arg) {
    // creamos un proceso hijo
    pid_t pid = fork();
    // reemplazamos el programa actual
    if (pid == 0) { // child process
        exec(arg);
    } else { // parent
        wait_for_child(pid);
    }
}
```
